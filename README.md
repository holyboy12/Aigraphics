# 🎨 AI Graphics Generator

An AI-powered web application that generates graphic designs from user prompts.  
Users can describe a design in natural language, and the system produces a visual output using generative AI.

---

## 🚀 Features

- 🧠 Text-to-image generation using AI
- 💬 ChatGPT-style input interface
- ⚡ Fast backend powered by FastAPI
- 🎨 Clean and modern UI
- 🔄 Real-time prompt processing

---

## 🛠️ Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript

**Backend**
- Python
- FastAPI

**AI Model**
- OpenAI Image Generation API

## ⚙️ How It Works

1. User enters a design prompt  
2. Frontend sends request to backend  
3. Backend processes prompt using AI model  
4. Generated image is returned and displayed
